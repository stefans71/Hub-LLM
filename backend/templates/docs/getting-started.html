Getting Started

First steps in your HubLLM workspace — launch Claude, describe a task, and start building.

1

Your Workspace

When you create a project, HubLLM sets up two directories on your VPS:

  • •Director — {slug}-director/ — manages the project: planning, task creation, and review.
  • •Engineer — {slug}/ — writes code, runs tests, and handles git.
Tip: The Director and Engineer each run their own Claude Code session in separate terminals.
2

Launch Your AI Engineer

Open the LLM-Dev Terminal panel in the lower-left of the screen. Copy and paste this command:

cd /root/llm-hub-projects/{slug} && claude

This starts a Claude Code session in your Engineer directory. Claude will read your codebase, write code, and commit changes.

3

Start the Project Director

Come back to the main terminal (upper area) and type:

claude

Press Enter. This starts your AI Coding session with the Project Director AI Persona.

Tip: Once the Director is running, type /generate-prp to create a detailed project brief that guides your AI team.
4

Terminal Tips

Copy: Highlight text in the terminal, right-click and select "Copy".

Paste: Right-click in the terminal and select "Paste".

Tip: HubLLM works with any terminal-based AI coding tool. Claude Code is the default.
5

Next Steps

Once both terminals are running, the Director will guide you through the rest of the workflow — task planning, execution, and review.

For more details, see the Workspace Guide or browse the full documentation.
