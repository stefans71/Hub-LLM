Feature Docs Portal

Launched the multi-page documentation portal. All HubLLM platform help lives in one searchable, navigable place.

  • Shared navigation sidebar with current-page highlighting
  • Text search on every page
  • Responsive design that works at any preview panel width

Fix GitHub OAuth Error Handling

GitHub OAuth popup now shows a clear error message instead of silently failing when the server-side GITHUB_CLIENT_ID isn't configured.

Removed all Codespaces messaging from the project creation wizard. GitHub is now positioned as version control and cloud backup. VPS card shows first as the primary hosting option.

The LLM-Dev panel now supports up to four split-pane terminals. Each terminal runs an independent SSH session.
