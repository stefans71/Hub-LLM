Getting Started

You're in your HubLLM workspace — everything you need to build with AI is right here. Follow these steps to start your first task:

  1. 1. In the terminal (center panel), cd into your project directory. It may have already done this for you.
  2. 2. Type claude to start your AI engineer, or claude --resume to continue a previous session.
  3. 3. Describe what you want to build — a feature, a fix, or paste a task description. Claude will write the code.
Tip: Claude commits after each task automatically. You can review changes in the File Explorer on the left.

Git & Version Control

Your project is already tracked with Git — git init happened when the project was scaffolded.

What are commits?

A commit is a snapshot of your code at a point in time. Think of it like a save point in a game — you can always go back to any previous commit. Each commit has a message describing what changed.

Claude automatically creates commits after completing each task, so your work is always saved with clear descriptions of what was done.

Connect to GitHub (optional)

GitHub is a cloud service that stores your code online. Connecting gives you cloud backup, version history, and the ability to share your project.

Sign up for GitHub Sign in to GitHub

Push your code to GitHub

After creating a repository on GitHub, connect it:

# Add your GitHub repo as a remote
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git

# Push your code
git push -u origin main
