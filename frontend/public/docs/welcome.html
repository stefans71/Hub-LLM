Getting Started

Your first steps in the HubLLM workspace — from opening your project to building with AI.

1

Launch Your AI Engineer

You're in your HubLLM workspace — everything you need to build with AI is right here. Follow these steps to start your first task:

  1. 1. In the terminal (center panel), cd into your project directory. It may have already done this for you.
  2. 2. Type claude to start your AI engineer, or claude --resume to continue a previous session.
  3. 3. Describe what you want to build — a feature, a fix, or paste a task description. Claude will write the code.
Tip: Claude commits after each task automatically. You can review changes in the File Explorer on the left.

3

Quick Tips

Here are some helpful things to know:

  • • README.md — Open it from the File Explorer for project-specific setup info.
  • • Hint bubble — The floating bubble in the terminal has quick command tips.
  • • Ask Claude — Claude can explain code, debug errors, or help you learn. Just describe what you need.
  • • CLAUDE.md — This file in your project root has rules and patterns Claude follows automatically.
Tip: You can type any URL in the Preview panel's address bar to preview websites while you build.
