What's New

Changelog and release notes — new features, improvements, and fixes.

February 9, 2026
Feature Docs Portal

Launched the multi-page documentation portal you're reading right now. All HubLLM platform help lives in one searchable, navigable place.

  • Six pages covering getting started, workspace, Git, changelog, and the harness system
  • Shared navigation sidebar with current-page highlighting
  • Text search on every page
  • Responsive design that works at any preview panel width
February 9, 2026
Feature VibeShip CTA Button

Added a VibeShip call-to-action button next to the PRP download in the OpenRouter setup flow. Users who need a VPS can quickly navigate to VibeShip for cloud hosting.

February 9, 2026
Fix GitHub OAuth Error Handling

GitHub OAuth popup now shows a clear error message instead of silently failing when the server-side GITHUB_CLIENT_ID isn't configured. The frontend pre-checks the /api/auth/providers endpoint before opening the OAuth popup.

February 9, 2026
Improvement Create Project Reframe

Removed all Codespaces messaging from the project creation wizard. GitHub is now positioned as version control and cloud backup (which is what it actually is). VPS card shows first as the primary hosting option.

February 9, 2026
Feature Welcome Page

New welcome page auto-loads in the Preview Panel on first project open. Includes getting started steps, workspace diagram, Git setup guide, and helpful tips. Shows once per project, then gets out of your way.

February 9, 2026
Feature Preview Panel Drag Resize

The Preview Panel now has a draggable left edge for resizing. No more fixed width — give the preview as much or as little space as you need.

January 2026
Feature Multi-Terminal Split Panes

The LLM-Dev panel now supports up to four split-pane terminals. Each terminal runs an independent SSH session, so you can run a dev server in one pane and tail logs in another.

January 2026
Feature Claude Code Terminal Chat

Integrated Claude Code directly into the workspace terminal. Start a session with claude and interact with your AI engineer without leaving HubLLM. Includes bubble view for a chat-like experience.

January 2026
Feature Monaco Code Editor

Added a VS Code-powered code editor in the LLM-Dev panel. Click any file in the File Explorer to open it with full syntax highlighting, IntelliSense hints, and multi-tab support.

December 2025
Feature VPS Connection Manager

Link a VPS to your project for a persistent development environment. SSH connections are managed automatically with reconnection on drop. Supports password and key-based auth.

December 2025
Feature Platform Launch

HubLLM.dev goes live — AI-powered development workspaces with Claude Code integration, project management, and a full web-based IDE. Powered by VibeShip.
