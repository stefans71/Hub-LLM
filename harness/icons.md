## Notes

- All icons use Feather Icons style (stroke-based)
- Default stroke width is 2
- Use `strokeWidth="3"` for bolder icons (like the + button)
- Add `style={{ opacity: 0.7 }}` for muted icons
- Use `style={{ flexShrink: 0 }}` when icon is in a flex container
